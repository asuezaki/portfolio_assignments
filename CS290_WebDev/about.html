Projects I Am Working On

Xiangqi

  • Python implementation of the game Xiangqi/Chinese Chess
  • The game ends when one player checkmates the other general
  • Board specified using algebraic notation: columns (a-i) and rows (1-10)
  • Row 1 is the red side and row 10 is the black side
  • Game rules can be found here

MASM x86 Project

  • Implementing low-level I/O procedures and macros in MASM x86
  • Small test program that gets 10 valid integers from the user and stores the numeric values in an array and then displays the integers, their sum, and their average.
  • Implements and tests ReadVal and WriteVal procedures for signed integers
  • Numeric input read as a string and converted to numeric form with input validation

FBoard

  • Python implementation of the game FBoard
  • Player x must get their piece to row 7
  • Player o must try to make it so player x has no legal moves
  • 8x8 game board initialized as a list of lists
  • Features input validation for legal moves
